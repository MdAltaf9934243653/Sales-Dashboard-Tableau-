# 📊 Sales Dashboard (Tableau)

## 🚀 Project Overview
This project presents an interactive Sales Dashboard built using Tableau to analyze sales performance across different categories, regions, and products.

## 📌 Key Insights
- Sales distribution by Category, Region, and City
- Revenue analysis by Sub-Category
- Product-level performance tracking
- Sales vs Quantity trends

## 🛠 Tools Used
- Tableau
- Data Visualization
- Business Intelligence
